Each recipe is now stored in its own JSON file!

Step 1: Create Your Recipe File

Copy recipes/recipe-template.json to your category folder:

recipes/desserts/my-recipe.json
recipes/beverages/my-recipe.json
recipes/gravy/my-recipe.json
recipes/rice/my-recipe.json
recipes/breads/my-recipe.json
recipes/sauces/my-recipe.json

Step 2: Fill in the Recipe Details

Edit your JSON file with this structure:

{
    "id": "your-recipe-id",
    "title": "Your Recipe Name",
    "category": "desserts",
    "description": "A brief, appetizing description",
    "prepTime": "15 mins",
    "cookTime": "30 mins",
    "servings": "4",
    "featured": false,
    "images": [
        "recipes/category/recipe-id/image1.jpg"
    ],
    "ingredients": [
        "1 cup ingredient one",
        "2 tablespoons ingredient two"
    ],
    "instructions": [
        "Step 1: First instruction",
        "Step 2: Second instruction"
    ],
    "notes": "Optional tips and notes"
}

Step 3: Register Your Recipe

Open scripts/config.js and add to the recipeFiles array:

recipeFiles: [
    'desserts/chocolate-cake',
    'desserts/your-recipe-id',  â† Add this line
    'beverages/mango-lassi',
    // ... other recipes

Step 4: Add Images (Optional)

  1. Create folder: recipes/category/your-recipe-id/
  2. Add images: image1.jpg, image2.jpg
  3. Update JSON: "images": ["recipes/category/your-recipe-id/image1.jpg"]

ğŸ“‹ Required Fields

  • id: Unique identifier (e.g., "chocolate-chip-cookies")
  • title: Recipe name
  • category: desserts, beverages, gravy, rice, breads, or sauces
  • description: Short description (1-2 sentences)
  • prepTime: Preparation time
  • cookTime: Cooking time
  • servings: Number of servings
  • ingredients: Array of ingredient strings
  • instructions: Array of step-by-step instructions

  • featured: Show on home page (true/false)
  • images: Array of image URLs or paths
